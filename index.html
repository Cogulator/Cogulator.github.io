Cogulator is a simple human performance calculator developed by MITRE for estimating task time and difficulty (i.e., workload).

GETTING STARTED

If you’re ready to start modeling, getting Cogulator installed on a Mac or PC is easy.

OPEN SOURCE

Cogulator is provided gratis under an Apache 2.0 license. Feel free to look under the hood. If you’d like to contribute to the project, create a pull request. Cogulator is built using HTML, CSS, Javascript, and Electron.

GET IN TOUCH

Let us know if you run into any bugs. For more general questions, feel free to get in touch with the project lead, Steven Estes, at sestes@mitre.org

NEWS

. (8/25/2020) v4.3.3 of Cogulator has been released. Fixed a bug that caused only part of a referenced goal method to be copied if there was a comment or blank link in the method.

. (8/24/2020) v4.3.1 of Cogulator has been released. Can now drag and drop and model into a new folder in the models sidebar. Several bug fixes for dark mode.

. (8/19/2020) v4.3.0 of Cogulator has been released. Cogulator will now remember your settings for dark mode, line numbers, and side bar width between sessions. Numerous bug fixes.

. (8/6/2020) v4.2.4 of Cogulator has been released. Fixed a series of bugs, including not being able to select a model and dark mode getting turned off when resizing the sidebar window.

. (8/5/2020) v4.2.3 of Cogulator has been released. You can now resize the models sidebar. Fixed bug in Windows that prevented scrolling to the bottom of the models list in the sidebar.

. (7/30/2020) v4.2.2 of Cogulator has been released. You can now double click on a model name to rename it. Fixed a bug that caused line numbers and error/tip markers to be misplaced. If you're wondering, 4.1.1 was not released externally.

. (5/22/2020) v4.2.0 of Cogulator has been released. Now has a Dark Mode (View -> Dark). Includes bug fixes.
