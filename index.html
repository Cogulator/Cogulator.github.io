Cogulator is a simple human performance calculator developed by MITRE for estimating task time and difficulty (i.e., workload).

GETTING STARTED

If you’re ready to start modeling, getting Cogulator installed on a Mac or PC is easy.

1. Install Adobe AIR 2. Install Cogulator

The whole process shouldn’t take more than 2 or 3 minutes. Let us know if you have questions or run into any bugs.

OPEN SOURCE

Cogulator is provided gratis under an Apache 2.0 license. Feel free to look under the hood. If you’d like to contribute to the project, create a pull request. Cogulator is developed in Flash CS6 and deployed as an Air application.
