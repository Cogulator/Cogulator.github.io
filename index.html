Cogulator is a simple human performance calculator developed by MITRE for estimating task time and difficulty (i.e., workload).

GETTING STARTED

If you’re ready to start modeling, getting Cogulator installed on a Mac or PC is easy.

OPEN SOURCE

Cogulator is provided gratis under an Apache 2.0 license. Feel free to look under the hood. If you’d like to contribute to the project, create a pull request. Cogulator is built using HTML, CSS, Javascript, and Electron.

GET IN TOUCH

Let us know if you run into any bugs. For more general questions, feel free to get in touch with the project lead, Steven Estes, at sestes@mitre.org

NEWS

. (5/22/2020) v4.2.0 of Cogulator has been released. Now has a Dark Mode (View -> Dark). Includes bug fixes.

. (1/23/2020) v4.1.3 of Cogulator has been released. Minor bug fixes, code cleanup, and syntax highlight for "As" in goal statements.

. (11/14/2019) v4.1.2 of Cogulator has been released. Some bug fixes for @References.

. (6/17/2019) v4.1.1 of Cogulator has been released. Should make interacting with large models (400+ lines) a bit less laggy. In addition, fixed a Windows bug that was preventing model deletion and added Find & Toggle Line Number to the Windows menus.

. (1/25/2019) v4.1 of Cogulator has been released. Thanks to the folks from SoarTech, we've got a slew of improvements, including: Optional line numbers, Find capability, new interactivity in the Gantt chart, and more. In addition, we've added a new featured called @References.

. (7/3/2018) Cogulator has been rebuilt from the ground up and no longer requires Adobe AIR. Windows and Mac versions available. Linux users can build from source.
